## ✨ ویژگی‌ها | Features

- ✅ **تحلیل دوگانه افکار**: موتور محلی (Rule-based) برای استفاده آفلاین + هوش مصنوعی (OpenRouter) برای تحلیل عمیق
- ✅ **۶ اختلال اصلی**: الگوهای شناختی اختصاصی برای افسردگی، اضطراب، شخصیت مرزی، وسواس، PTSD و ADHD
- ✅ **بازسازی شناختی بلادرنگ**: تبدیل افکار ناسالم به افکار متعادل
- ✅ **تمرین‌های درمانی اختصاصی**: هر اختلال شامل تکنیک‌های استاندارد CBT و تمرین‌های عملی
- ✅ **هوش مصنوعی مولد**: ارائه راهکارهای شخصی‌سازی‌شده با OpenRouter (اختیاری)
- ✅ **ردیابی پیشرفت**: نمودار روند بهبودی و ذخیره تاریخچه جلسات
- ✅ **خروجی JSON/CSV**: امکان ذخیره و تحلیل داده‌ها
- ✅ **ارسال به پرونده درمانی**: یکپارچه‌سازی با  [دستیار روان‌درمانی]
- ✅ **طراحی واکنش‌گرا**: بهینه‌شده برای موبایل و دسکتاپ
- ✅ **حالت تاریک**: مناسب استفاده طولانی‌مدت
- ✅ **بدون نیاز به نصب**: فقط با مرورگر اجرا می‌شود
- ✅ **حریم خصوصی**: داده‌ها به‌صورت محلی در مرورگر ذخیره می‌شوند

## 🎯 اختلالات تحت پوشش | Covered Disorders

| اختلال | Disorder | الگوهای شناختی | تکنیک‌ها |
|--------|----------|----------------|-----------|
| 😔 افسردگی | Depression | تفکر همه یا هیچ، احساس بی‌ارزشی، درماندگی | فعال‌سازی رفتاری، بازسازی شناختی |
| 😰 اضطراب | Anxiety | پیش‌بینی فاجعه، تحمل‌ناپذیری، فکر چه‌می‌شود | تکنیک‌های تنفسی، مواجهه تدریجی |
| 🌀 شخصیت مرزی | BPD | تفکر سیاه و سفید، ترس از رهاشدگی | مهارت‌های DBT، تحمل پریشانی |
| 🔄 وسواس | OCD | شک وسواسی، کمال‌گرایی | ERP، پیشگیری از پاسخ |
| 💥 PTSD | PTSD | بازتجربه، اجتناب | زمین‌گیرسازی (Grounding) |
| ⚡ ADHD | ADHD | مشکل توجه، مدیریت زمان | ساختاردهی محیط، تایمر پومودورو |

## 🛠️ تکنولوژی‌ها | Tech Stack

| تکنولوژی | کاربرد |
|-----------|--------|
| **HTML5** | ساختار برنامه |
| **CSS3** | استایل‌دهی و طراحی واکنش‌گرا |
| **JavaScript (Vanilla)** | منطق برنامه، تحلیل الگوها، مدیریت داده‌ها |
| **OpenRouter API** | اتصال به مدل‌های هوش مصنوعی (GPT-3.5/4) |
| **localStorage** | ذخیره‌سازی محلی داده‌ها |
| **Canvas API** | رسم نمودار پیشرفت |

> **صفر وابستگی خارجی** - بدون نیاز به Node.js، npm، یا هر کتابخانه اضافی
